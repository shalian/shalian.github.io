宝塔部署 nodejs 项目

更新于2023-10-10
|字数总计:382|阅读时长:1分钟|阅读量:

前言

首先你已经有了服务器，并且打开了宝塔面板，其次准备好你的 nodejs 项目。
在宝塔安装 pm2管理器（推荐）

打开 pm2 设置，选择nodejs项目对应版本

在左侧菜单点击 文件，在根目录点击 www，其中 wwwroot 就是宝塔的默认建站目录

部署

  1. 新建目录 test ，进入文件夹，上传你的 nodejs 项目。上传 除 node_modules 外其他文件，然后打开宝塔内置 终端 安装依赖。右键文件夹空白处刷新即可。
    PS：务必使用对应的包管理工具，如果你使用了非 npm 工具，记得在 pm2 的 模块管理 中下载对应工具。

  2. 接下来就是放行端口，如果不放行是无法访问的，此处需要放行两处，宝塔面板的安全菜单以及云服务器的防火墙，我的项目端口用的 5555 那就放行 5555。

  • 宝塔面板

  • 云服务器
    进入云服务器 实例，点击 安全组，点击 安全组ID/名称，点击添加端口即可

  1. 返回宝塔面板，打开 pm2 添加项目，填入项目目录和启动文件，点击提交即可。

  2. 添加项目成功后会显示对应端口，并且状态是绿色播放图标，没有显示端口号的话肯定是项目启动失败了，建议重新走一遍流程。

  3. 接下来在浏览器输入 ip/域名:端口号 来访问 node 项目了
