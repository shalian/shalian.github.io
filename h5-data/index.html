HTML5的 data-* 属性

更新于2023-12-16|HTML
|字数总计:328|阅读时长:1分钟|阅读量:

data-* 属性简介

data-*全局属性是一类被称为自定义数据属性的属性，它赋予我们在所有 HTML 元素上嵌入自定义数据属性的能力。

简言之，就是可以传入自己定义的数据，因为之前 HTML 没有这个功能，用的话也是放在 css 里，但是这不是 css 该做的事，所以 HTML5 中加入了这个属性，就能把自定义的变量放在 data-*

data-*的使用

在 HTML 里边用 data-变量名使用，如下：
<div class="box" data-name="Tom" data-age="18"></div>

在 JS 中有两种获取方法

  1. dataset：所有放在 data-*中的属性都在 dataset 中
  1. getAttribute() 方法获取

setAttribute() 使用

使用 setAttribute() 可以设置/修改属性值

CSS 中使用

在 css 中可以使用 .类名[data-*=属性值] 的方式找到对应的元素

伪元素中使用

在 before/after 伪元素中可以使用 attr(data-\*) 的方式获取到对应的属性值
