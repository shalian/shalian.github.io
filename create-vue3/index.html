下载 nodejs

使用 create vite 前提：node js版本建议大于16.0（查看版本命令：node -v 执行命令。下载 nodejs

  1. 配置环境变量

环境变量的作用是系统运行一个程序而没有告诉它程序所在的完整路径时，系统除了在当前目录下面寻找此程序外，还应到 path 中指定的路径去找。用户通过设置环境变量，来更好的运行进程。

打开控制面板 ——> 搜索高级系统设置 ——> 环境变量，在环境变量中找到 Path，点击编辑，点击新建，输入 nodejs安装目录\nodejs\

初始化

使用 npm、yarn、pnpm 其中一种命令安装即可，我这里使用 pnpm。

pnpm 是集合了 npm 和 yarn 优点的，在使用前需要先全局安装 npm install pnpm -g

  • 使用 npm
  • 使用 pnpm
  • 使用 yarn
  • 执行 pnpm create vite@latest ，输入 y 按回车，
  • 输入项目名称(例如 demo)，按回车键，
  • 选择 Vue (也可以选择 React、Vanilla 等其他框架)，按回车
  • 可自行选择使用 js 还是 ts，按回车键就大功告成了。
