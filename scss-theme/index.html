主题切换

定义多种 :root 主题颜色变量

sass 定义多种 :root 主题颜色变量

下面例子中，我们定义了两套主题色，并通过更改 $theme-color 变量来切换不同的主题。然后，我们使用 :root 选择器将主题色定义为 CSS 变量。在实际使用中，可以通过 JavaScript 动态更改这些 CSS 变量来实现主题的实时切换。并设置不同主题颜色。

注：这种方法只能手动切换 $theme-color 的值(这些变量是在编译时确定的，不是动态的)，但可以使用 js 修改 :root 中的值。

使用时，可以通过 JavaScript 更改主题色，例如，使用 localStorage 记住用户选择
