数组 Api

更新于2023-11-26|数组 Api
|字数总计:1.5k|阅读时长:5分钟|阅读量:

改变原数组

pop()、push()、shift()、unshift()、reverse()、sort()、splice()

pop()

pop() 方法用于删除数组的最后一个元素并返回删除的元素。
PS：移除数组第一个元素，请使用 shift() 方法。

push()

push() 方法可向数组的末尾添加一个或多个元素，并返回新的长度。
PS： 在数组起始位置添加元素请使用 unshift() 方法。

shift()

shift() 方法用于把数组的第一个元素从其中删除，并返回第一个元素的值。
PS：移除数组末尾的元素可以使用 pop() 方法。

unshift()

unshift() 方法用于把数组的第一个元素从其中删除，并返回第一个元素的值。
PS：在数组末尾添加一项，请使用 push() 方法。

reverse()

reverse() 方法用于颠倒数组中元素的顺序。

sort()

sort() 方法用于对数组的元素进行排序。
排序顺序可以是字母或数字，并按升序或降序。默认排序顺序为按字母升序。

当数字是按字母顺序排列时 “40” 将排在 “5” 前面。
使用数字排序，你必须通过一个函数作为参数来调用。
函数指定数字是按照升序还是降序排列。例如：arr.sort((a, b) => a - b)

splice()

splice() 方法用于添加、修改或删除数组中的元素。如果删除一个元素，则返回删除的元素组成的数组。 如果未删除任何元素，则返回空数组。

有三个参数：

  • index —— 必需。整数，规定添加/删除项目的位置，使用负数可从数组结尾处规定位置。
  • howmany —— 必需。要删除的项目数量。如果设置为 0，则不会删除项目。
  • item1, …, itemX —— 可选。向数组添加的新项目。

详情

不改变原数组

reduce()、concat()、every()、some()、filter()、find()、map()、forEach()、slice()、join()

reduce()

reduce() 方法接收一个函数作为累加器，数组中的每个值（从左到右）开始缩减，最终计算为一个值。
接收两个参数：

  • 第一个参数：每次遍历都会调用的函数，而这个函数有接收四个参数，分别是：前一个值、当前项、项目的索引和数组对象，而这个函数的返回值，回传给下一次遍历时，执行的这个方法的第一个参数。
  • 第二个参数：归并基础的初始值

语法：arr.reduce((prev, cur, index, arr) => {}, [])
详情。
PS：reduce() 对于空数组是不会执行回调函数的

map()

map() 方法返回一个新数组，数组中的元素为原始数组元素调用函数处理后的值。
语法：arr.map((item, index) => { return item })
PS：map() 不会对空数组进行检测。

forEach()

forEach() 方法用于调用数组的每个元素，并将元素传递给回调函数。
forEach() 没有返回值，而且它也不会改变原数组。

slice()

slice(start, end) 方法可从已有的数组中返回指定的元素。
可提取字符串的某个部分，并将提取的部分以新的字符串返回。

  • start 是一个从 0 开始的索引，用于开始复制数组的一部分。如果未定义，start 的默认值为 0。如果 start 大于数组的索引范围， slice() 方法将返回一个空数组。
  • end 参数是可选的。如果 slice() 函数中只有一个参数，那就是 start。如果省略， slice() 方法从数组的末尾开始提取，不包括索引 end 在内。如果 end 大于数组的长度，slice() 一直提取到数组的末尾，只是在它被省略的情况下。

join()

join() 方法用于把数组中的所有元素转换一个字符串，括号可选元素通过什么分隔符进行分隔。
语法：arr.join('')
PS：如果将字符串分割成数组，需使用 str.split(',')，括号可选择按什么字符分割

concat()

concat() 方法用于连接两个或多个数组。
语法：arr1.concat(arr2)

every()

every() 方法用于检测数组所有元素是否都符合指定条件（通过函数提供）。
语法：arr.every((item, index) => { return 条件 })
every() 方法使用指定函数检测数组中的所有元素：

  • 如果数组中检测到有一个元素不满足，则整个表达式返回 false ，且剩余的元素不会再进行检测。
  • 如果所有元素都满足条件，则返回 true。

PS：every() 不会对空数组进行检测。

some()

some() 方法用于检测数组中的元素是否满足指定条件（函数提供）。
语法：arr.some((item, index) => { return 条件 })
some() 方法会依次执行数组的每个元素：

  • 如果有一个元素满足条件，则表达式返回 true , 剩余的元素不会再执行检测。
  • 如果没有满足条件的元素，则返回 false。

PS：some() 不会对空数组进行检测。

filter()

filter() 方法筛选符合条件的数据，并返回一个新数组。
语法：arr.filter((item, index) => { return 条件 })
PS：filter() 不会对空数组进行检测。

find()

find() 方法筛选第一个符合条件的数据，并返回该数据，后续不在遍历。
语法：arr.find((item, index) => { return 条件 })
PS：find() 不会对空数组进行检测。
